# Protokoll Tokezimi – Swiss Approval Albania

Aplikacion web për gjenerimin e Protokollit të Matjes së Rezistencës së Tokëzimit.

## 🚀 Si të publikoni në GitHub Pages (hap pas hapi)

### 1. Krijoni një llogari GitHub
Shkoni te [github.com](https://github.com) dhe regjistrohuni falas.

### 2. Krijoni një repository të ri
- Klikoni butonin **"+"** → **"New repository"**
- Emri: `protokoll-tokezimi` (ose çfarë dëshironi)
- Zgjidhni **Public**
- Klikoni **"Create repository"**

### 3. Ngarkoni skedarin
- Klikoni **"uploading an existing file"**
- Tërhiqni skedarin `index.html` dhe lëshojeni
- Klikoni **"Commit changes"**

### 4. Aktivizoni GitHub Pages
- Shkoni te **Settings** → **Pages**
- Tek *Source*, zgjidhni **"Deploy from a branch"**
- Branch: **main**, Folder: **/ (root)**
- Klikoni **Save**

### 5. Lidhja juaj është gati!
Pas 1-2 minutash, faqja do të jetë aktive te:
```
https://EMRI_JUAJ.github.io/protokoll-tokezimi/
```
